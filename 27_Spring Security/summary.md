### Struktur folder MVC
- Config : Kumpulan konfigurasi, Security, Bean
- Controller : sebagai kumpulan routing RestAPI atau sering disebut endpoint.
- Model/Entity : Bentuk dasar dari business model , POJO (Plain Old Java Object)
- Repository : Sebagai interface yang berhubungan langsung dengan database.
- Security : Penyedia token JWT, dan security filter.
- Service : Kumpulan interface digunakan untuk business process/ rules/ logic.
